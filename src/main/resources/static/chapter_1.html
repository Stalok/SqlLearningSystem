SQL 簡介

SQL（Structured Query Language，結構化查詢語言）是一種用於管理關聯式資料庫管理系統（RDBMS）的計算機語言。它允許你存儲、操作和檢索存儲在關聯數據庫中的數據。

SQL 的用途

SQL 的範圍包括以下方面：

  1. 數據操作：
    • 查詢：使用 SELECT 語句從數據庫中檢索數據。
    • 插入：使用 INSERT INTO 語句將新記錄插入數據庫。
    • 更新：使用 UPDATE 語句修改數據庫中的數據。
    • 刪除：使用 DELETE FROM 語句從數據庫中刪除記錄。
  2. 數據庫模式和設計：
    • 創建和修改表格：使用 CREATE TABLE 和 ALTER TABLE 語句來設計和修改數據表。
    • 數據庫模式：設計數據庫的結構，包括表之間的關係。
  3. 數據訪問控制：
    • 權限設置：使用 SQL 可以設置表、存儲過程和視圖的權限。

SQL 的標準

  • SQL 在 1986 年成為美國國家標準化組織（ANSI）的一項標準，並在 1987 年成為國際標準化組織（ISO）的標準。
  • 儘管 SQL 是一門 ANSI 標準的計算機語言，但仍然存在多種不同版本的 SQL 語言。為了與 ANSI 標準相兼容，它們必須以相似的方式共同地支持一些主要的命令，例如 SELECT、UPDATE、DELETE、INSERT、WHERE 等等。

使用 SQL

要在你的網站中使用 SQL，你需要：

  1. RDBMS 數據庫程序：例如 MySQL、MS Access、SQL Server 等。
  2. 伺服器端腳本語言：例如 PHP 或 ASP。
  3. HTML / CSS：用於顯示數據。

RDBMS（關聯式數據庫管理系統）是 SQL 的基礎，也是所有現代數據庫系統的基礎，例如 MS SQL Server、IBM DB2、Oracle、MySQL 和 Microsoft Access。在 RDBMS 中，數據存儲在稱為表的數據庫對象中，表由列和行組成。
